# JackBuffer
二进制数据序列化协议, 含自动生成接口

## 依赖
JackAST: https://github.com/chenwansal/JackAST  
BufferIO: https://github.com/chenwansal/BufferIO

## 支持
byte, byte[]  
sbyte, sbyte[]  
ushort, ushort[]  
short, short[]  
uint, uint[]  
int, int[]  
ulong, ulong[]  
long, long[]  
char, char[]  
string, string[](utf-8)  
float, float[]  
double, double[]  

## 暂不支持
自定义类型, 自定义类型[]  
枚举, 枚举[]
